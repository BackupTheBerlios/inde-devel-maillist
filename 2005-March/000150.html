Normally, when you're commiting your work the commit will include the 
API docs in doc/ as well.
But of course this won't update the API docs linked on the website. So I 
made the website
directory api/ a local working copy of trunk/doc/. This means one can 
easily sync the ones
on the website with those on SVN by typing "svn update" (after logging 
in on shell.berlios.de
and switching to the directory /home/groups/inde/htdocs/api/).

This has nothing to do with what you do with the local working copy on 
your home computer.
