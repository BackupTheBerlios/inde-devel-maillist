Hi Jochen.

I finally got my stuff commited to SVN. I removed your dialogs since the 
current SVN trunk
wouldn't build anymore with them (and I didn't want to do your work by 
changing them). You
may of course integrate them again.

Some notes (besides the API docs you can look at in doc/index.html) on 
the most
important API changes:

Class: InDE (previously called "App")
    - removed syntax file loading functionality (is now handled by 
TabEditor)

Class: MainWindow
    - all onCmd... and onUpd... functions
    + a public member variable "settings" which is an object of the 
class Settings
    > onCmdEditSettings() to editSettings()
    + methods applySettings(), resetSettings()
    + method readSettings() - default values are defined here
    + methods to build menus and content: buildMenu(), buildContent()
    + variable "editor" (class TabEditor) and method getEditor()
    > settings file is now created in ~/.InDE/. It's called "settings" 
without
      any extension on linux. On Windows, it is created as 
~/InDE/settings.ini
    ...

Class: SettingsDialog
    + method run() to execute the dialog
    + methods loadValues() and storeValues() (that's no longer handled 
by MainWindow)
    + some options

Class: Settings
    + All methods are written in two ways: with and without section 
parameter. The ones
      without a section parameter use section "GENERAL"
    + methods: setDefault() for integer and string values.
    + methods: getStringValue(), getIntValue() and getFormatValue() (the 
latter probably
      won't work already)
    + methods setStringValue, setIntValue() and setFormatValue() (the 
latter probably
      won't work)
    + methods parse(), read() and write()
    + method defaults(FXbool) for either filling missing values or 
filling and replacing
      all values with defaults defined using setDefault

Class: TabItem
    + Enhancement of FXTabItem. Can send messages to a target.

Class: Edit
    + represents an opened document/file
    + methods getFilename(), setFilename(), isModified() and setModified()
    + method setTitleTab() for connecting to a TabItem object which 
displays the
      file's basename
    + method setSyntax() for settings the syntax for the document

Class: TabEditor
    + tons of methods for loading a syntax file, determining the syntax 
of documents,
      opening, closing and saving one, more or all documents, and so on ...

Class: FileBrowser
    + Enhanced FXDirList. Provides context menus for files and 
directories like opening,
      importing and deleting.

I hope this helps. If you have any questions - ask them!

Kind regards,
Jannis
