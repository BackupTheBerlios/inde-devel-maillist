Hi Eugene.

>>Has the use of an XML format been considered for configuration file 
>>settings?
>>
 XML is really nice of course, but to be honest, reviewing and modifying 
an INI-file is much easier
than to do this with an XML file (INI has a much simpler syntax -> files 
are more concise).

>>You're using FXScintilla for the text editing component, or so I've read 
>>from archives, right?
>>
Yes, I thought about it. But it's not yet ported to FOX 1.4 (as far as I 
know). Another reason for not
choosing it was its "strange" messaging system. I prefer to change 
properties by specific
functions (like setFont or setForeColor) instead of using the same 
method for each change
just with a specific parameter (like SCI_STYLESETFONT).

Due to this, I chose to use a simple FXText combined with 
Syntaxhighlighting as it is used
in Adie (the editor written by FOX author Jeroen). We will see whether 
this way is good or
not.
